midPoint is an open standards based Identity Management, Provisioning and Compliance solution. Experience shows that the most important features of an identity management product are: high flexibility in Business Process handling and compliance with open standards and interfaces. A highly flexible user interface combined with a very robust workflow engine make midPoint ready for any Identity Management project.

The provisioning services of midPoint are open for several identity related components: Role Management, Identity Management, Compliance Calibration and others. With this kind of open architecture, the provisioning service can be used for all user-related data like user objects, as well as user group memberships or role memberships.

Key Features

  • User account livecycle handling
  • Role handling, both in openIDM and external systems and role based access control (RBAC)
  • Flexible resource connection architecture
  • Organizational hierarchy of all objects
  • Access control through through flexible capabilities
  • Object relation handling
  • Extended Auditing
  • Password Synchronization
  • Reconciliation
  • Pickup of external changes in near Real Time
